Pessoas cadastradas
💡 Como pegar o Slack User ID:
No Slack, clique no perfil da pessoa → Ver perfil → menu ⋯ → Copiar ID do membro.
O ID tem o formato U07EADY4SGN. Com ele a marcação @pessoa funciona de verdade na mensagem.
Grupos de pessoas (@)
💡 Como pegar o Slack Group ID:
No Slack, vá em Configurações do espaço de trabalho → Grupos de usuários. O ID do grupo tem formato S0614TZR7.
Ou use o handle do grupo (ex: @time-ti) — funciona para grupos de usuários existentes.
